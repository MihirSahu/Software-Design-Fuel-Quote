# COSC 4353 Project

This is the repository for Group 57's Software Design Project.

## npm / yarn scripts

To interact with the website:
- `npm run dev`
- 'yarn dev'

To run unit tests using Jest:
- 'npm run jest'
- 'yarn jest'

To run code coverage test using Jest:
- 'npm run jest:coverage'
- 'yarn jest:coverage'
